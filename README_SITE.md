# Fervidex Systems Static Website

## How to run locally

Open `index.html` directly in your browser, or run:

```bash
python -m http.server 8080
```

Then visit `http://localhost:8080`.

## Hosting

Upload all files to any static host: Netlify, Vercel, GitHub Pages, Cloudflare Pages, AWS S3 + CloudFront, or cPanel public_html.
