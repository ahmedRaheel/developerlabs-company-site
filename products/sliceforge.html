SliceForge Enterprise AI

SliceForge is a vertical slice architecture generator for modern .NET APIs.

It helps teams generate feature-first modules with Minimal APIs, CQRS commands, queries, validators, handlers, DTOs, endpoint mapping, tests, and clean dependency boundaries.

Core Features

  • Vertical Slice Architecture
  • Feature-first folders
  • CQRS command/query generation
  • Minimal API endpoint generation
  • MediatR/Mediator-ready handlers
  • FluentValidation integration
  • Clean Architecture rules inside every slice
  • EF Core write operations
  • Dapper read models
  • Outbox-ready templates
  • API-first structure
  • Swagger/Scalar-friendly endpoints
  • Unit and integration test scaffolding

Quick Start

bash
sliceforge new-api --name EStore --namespace EStore --output "D:\Projects\EStore"

sliceforge add-slice --module Catalog --feature CreateProduct --type command --properties "Name:string,Sku:string,Price:decimal"

sliceforge add-slice --module Catalog --feature GetProductById --type query --properties "Id:Guid"

sliceforge add-crud --entity Product --module Catalog --properties "Name:string,Sku:string,Price:decimal"

Example Command

csharp
public sealed record CreateProductCommand(
    string Name,
    string Sku,
    decimal Price) : IRequest<Guid>;

Example Endpoint

csharp
app.MapPost("/api/catalog/products", async (
    CreateProductCommand command,
    ISender sender) =>
{
    var id = await sender.Send(command);
    return Results.Created($"/api/catalog/products/{id}", new { id });
});
