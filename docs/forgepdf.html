ForgePDF AI

Enterprise-grade AI-powered PDF rendering and document conversion engine for .NET.

ForgePDF AI supports:

  • HTML to PDF
  • HTML Parts to PDF
  • Word (.docx) to PDF
  • Excel (.xlsx) to PDF
  • PowerPoint (.pptx) to PDF
  • Multilingual rendering
  • RTL language support
  • Large file rendering
  • Parallel rendering
  • Streaming PDF generation
  • Enterprise concurrency handling

Installation

Clone

bash
git clone https://github.com/your-repo/ForgePDF.git
cd ForgePDF

Build

bash
dotnet build

Run Sample API

bash
dotnet run --project samples/Enterprise.MinimalApi

Minimal API Example

csharp
using ForgePDF.Core;
using ForgePDF.Threading;
using ForgePDF.AspNetCore;

var builder = WebApplication.CreateBuilder(args);

builder.Services.AddForgePdf(options =>
{
    options.MaxConcurrentDocuments = Environment.ProcessorCount;
    options.QueueTimeout = TimeSpan.FromSeconds(30);
});

var app = builder.Build();

app.MapPost("/convert/html-to-pdf", async (
    HtmlToPdfRequest request,
    DocumentConversionService converter) =>
{
    var result = await converter.ConvertHtmlAsync(request);

    return Results.File(
        result.Bytes,
        "application/pdf",
        result.FileName);
});

app.Run();

HTML to PDF Example

csharp
var result = await converter.ConvertHtmlAsync(new HtmlToPdfRequest
{
    Html = """
    <html>
        <body>
            <h1>ForgePDF AI</h1>
            <p>Enterprise HTML to PDF rendering.</p>
        </body>
    </html>
    """,
    FileName = "report.pdf"
});

HTML Parts Example

csharp
var result = await converter.ConvertHtmlPartsAsync(
    new HtmlPartsToPdfRequest
    {
        HtmlParts =
        [
            "<h1>Part 1</h1>",
            "<h2>Part 2</h2>"
        ],
        CssParts =
        [
            "body { font-family: Arial; }"
        ],
        FileName = "parts.pdf"
    });

Word to PDF Example

csharp
var bytes = await File.ReadAllBytesAsync("resume.docx");

var result = await converter.ConvertFileAsync(
    new FileToPdfRequest
    {
        Kind = ForgeDocumentKind.WordDocx,
        Base64File = Convert.ToBase64String(bytes),
        OutputFileName = "resume.pdf"
    });
